# Sencilla.Web

ASP.NET Core web infrastructure for the [Sencilla Framework](https://github.com/vitalykovalgit/Sencilla).

## What's Included

- **CRUD controller scaffolding** — generate full REST APIs from entity definitions
- **Model binding** — custom binders for filter, sort, and pagination
- **Convention-based routing** — automatic route generation from entity names
- **API contracts** — standardized request/response models
- **Result types** — structured API responses

## Installation

```bash
dotnet add package Sencilla.Web
```
