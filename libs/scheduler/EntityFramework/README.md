# Sencilla.Scheduler.EntityFramework

Entity Framework Core persistence for the [Sencilla Framework](https://github.com/vitalykovalgit/Sencilla) scheduler. Store scheduled task definitions and execution history in a database via EF Core.

## Installation

```bash
dotnet add package Sencilla.Scheduler.EntityFramework
```
