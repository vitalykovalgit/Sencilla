# Sencilla.Component.Files.AmazonS3

Amazon S3 file storage provider for the [Sencilla Framework](https://github.com/vitalykovalgit/Sencilla). Store and retrieve files using Amazon S3.

## Installation

```bash
dotnet add package Sencilla.Component.Files.AmazonS3
```
