# Sencilla.Component.Files.GoogleStorage

Google Cloud Storage file provider for the [Sencilla Framework](https://github.com/vitalykovalgit/Sencilla). Store and retrieve files using Google Cloud Storage.

## Installation

```bash
dotnet add package Sencilla.Component.Files.GoogleStorage
```
