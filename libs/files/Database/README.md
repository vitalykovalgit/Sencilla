# Sencilla.Component.Files.Database

Database file storage provider for the [Sencilla Framework](https://github.com/vitalykovalgit/Sencilla). Store and retrieve files in a database using Entity Framework Core.

## Installation

```bash
dotnet add package Sencilla.Component.Files.Database
```
