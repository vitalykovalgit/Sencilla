# Sencilla.Component.Files.AzureStorage

Azure Blob Storage file provider for the [Sencilla Framework](https://github.com/vitalykovalgit/Sencilla). Store and retrieve files using Azure Blob Storage.

## Installation

```bash
dotnet add package Sencilla.Component.Files.AzureStorage
```
