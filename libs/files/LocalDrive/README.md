# Sencilla.Component.Files.LocalDrive

Local file system storage provider for the [Sencilla Framework](https://github.com/vitalykovalgit/Sencilla). Store and retrieve files on the local disk.

## Installation

```bash
dotnet add package Sencilla.Component.Files.LocalDrive
```
