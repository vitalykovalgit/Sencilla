# Sencilla.Repository.HttpClient

Remote API repository implementation via HttpClient for the [Sencilla Framework](https://github.com/vitalykovalgit/Sencilla). Access remote REST APIs using the same repository interfaces as local data access.

## Installation

```bash
dotnet add package Sencilla.Repository.HttpClient
```
