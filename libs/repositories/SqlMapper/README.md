# Sencilla.Repository.SqlMapper

Raw SQL repository implementation for the [Sencilla Framework](https://github.com/vitalykovalgit/Sencilla). Execute raw SQL queries while maintaining the Sencilla repository interface contract.

## Installation

```bash
dotnet add package Sencilla.Repository.SqlMapper
```
