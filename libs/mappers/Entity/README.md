# Sencilla.Mapper.Entity

Entity-to-entity mapping utilities for the [Sencilla Framework](https://github.com/vitalykovalgit/Sencilla). Map between domain entities, DTOs, and view models.

## Installation

```bash
dotnet add package Sencilla.Mapper.Entity
```
