# Sencilla.Mapper.Excel

Excel mapping utilities for the [Sencilla Framework](https://github.com/vitalykovalgit/Sencilla). Import and export data between Excel files and Sencilla entities.

## Installation

```bash
dotnet add package Sencilla.Mapper.Excel
```
