# Sencilla.Mapper.Sql

SQL mapping utilities for the [Sencilla Framework](https://github.com/vitalykovalgit/Sencilla). Map raw SQL query results to Sencilla entities.

## Installation

```bash
dotnet add package Sencilla.Mapper.Sql
```
