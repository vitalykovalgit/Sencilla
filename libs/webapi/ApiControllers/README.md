# Sencilla.Web.ApiControllers

API controller abstractions for the [Sencilla Framework](https://github.com/vitalykovalgit/Sencilla). Provides base CRUD controllers that automatically wire up to Sencilla repositories.

## Installation

```bash
dotnet add package Sencilla.Web.ApiControllers
```
