# Sencilla.Messaging.InMemoryQueue

In-process message queue for the [Sencilla Framework](https://github.com/vitalykovalgit/Sencilla). Ideal for development, testing, and single-instance scenarios where an external message broker is not needed.

## Installation

```bash
dotnet add package Sencilla.Messaging.InMemoryQueue
```
