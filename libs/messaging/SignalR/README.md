# Sencilla.Messaging.SignalR

SignalR (WebSocket) provider for the [Sencilla Framework](https://github.com/vitalykovalgit/Sencilla) messaging system. Push real-time messages to connected clients using SignalR as the transport.

## Installation

```bash
dotnet add package Sencilla.Messaging.SignalR
```
