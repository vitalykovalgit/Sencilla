# Sencilla.Messaging.Scheduler

Scheduled message dispatch for the [Sencilla Framework](https://github.com/vitalykovalgit/Sencilla). Schedule messages to be dispatched at specific times or on cron-based intervals.

## Installation

```bash
dotnet add package Sencilla.Messaging.Scheduler
```
