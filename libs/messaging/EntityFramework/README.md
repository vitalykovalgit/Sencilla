# Sencilla.Messaging.EntityFramework

Entity Framework Core persistence for the [Sencilla Framework](https://github.com/vitalykovalgit/Sencilla) messaging system. Store and process messages using EF Core as the backing store.

## Installation

```bash
dotnet add package Sencilla.Messaging.EntityFramework
```
