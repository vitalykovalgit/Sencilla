# Sencilla.Messaging.Kafka

Apache Kafka provider for the [Sencilla Framework](https://github.com/vitalykovalgit/Sencilla) messaging system. Publish and consume messages using Kafka as the transport.

## Installation

```bash
dotnet add package Sencilla.Messaging.Kafka
```
