# Sencilla.Messaging.Mediator

MediatR integration for the [Sencilla Framework](https://github.com/vitalykovalgit/Sencilla) messaging system. Dispatch messages in-process using MediatR as the transport.

## Installation

```bash
dotnet add package Sencilla.Messaging.Mediator
```
