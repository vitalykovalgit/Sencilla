# Sencilla.Messaging.ServiceBus

Azure Service Bus provider for the [Sencilla Framework](https://github.com/vitalykovalgit/Sencilla) messaging system. Publish and consume messages using Azure Service Bus as the transport.

## Installation

```bash
dotnet add package Sencilla.Messaging.ServiceBus
```
