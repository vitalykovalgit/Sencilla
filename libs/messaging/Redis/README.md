# Sencilla.Messaging.Redis

Redis Pub/Sub provider for the [Sencilla Framework](https://github.com/vitalykovalgit/Sencilla) messaging system. Publish and consume messages using Redis as the transport.

## Installation

```bash
dotnet add package Sencilla.Messaging.Redis
```
