# Sencilla.Component.Validation

Validation component for the [Sencilla Framework](https://github.com/vitalykovalgit/Sencilla). Provides validation rules, validators, and integration with the Sencilla command pipeline.

## Installation

```bash
dotnet add package Sencilla.Component.Validation
```
