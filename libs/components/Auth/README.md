# Sencilla.Component.Users.Auth

Authentication component for the [Sencilla Framework](https://github.com/vitalykovalgit/Sencilla). Provides JWT authentication, token generation, and identity integration.

## Installation

```bash
dotnet add package Sencilla.Component.Users.Auth
```
