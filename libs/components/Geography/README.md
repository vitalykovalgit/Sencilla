# Sencilla.Component.Geography

Geography component for the [Sencilla Framework](https://github.com/vitalykovalgit/Sencilla). Provides entities and utilities for working with countries, regions, and geographic data.

## Installation

```bash
dotnet add package Sencilla.Component.Geography
```
