# Sencilla.Component.I18n

Internationalization (i18n) component for the [Sencilla Framework](https://github.com/vitalykovalgit/Sencilla). Provides localization support and multi-language content management.

## Installation

```bash
dotnet add package Sencilla.Component.I18n
```
