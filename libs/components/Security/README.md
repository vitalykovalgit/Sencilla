# Sencilla.Component.Security

Authorization and roles component for the [Sencilla Framework](https://github.com/vitalykovalgit/Sencilla). Provides role-based access control, permission management, and authorization utilities.

## Installation

```bash
dotnet add package Sencilla.Component.Security
```
